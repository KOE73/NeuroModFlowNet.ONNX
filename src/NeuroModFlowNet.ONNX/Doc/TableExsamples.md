
Для создания профессиональных таблиц в консоли лучше всего использовать символы Unicode(Box Drawing Characters). Они идеально стыкуются друг с другом, образуя неразрывные линии.

Вот полный набор, разделенный по стилям:

1. Одинарные тонкие линии (Классика) ┌ ┐ └ ┘ ─ │ ┬ ┴ ├ ┤ ┼          ╭ ╮ ╰ ╯

2. Двойные линии(Для акцентов)       ╔ ╗ ╚ ╝ ═ ║ ╦ ╩ ╠ ╣ ╬	╤ ╧ ╟ ╢

4. Жирные(толстые) линии             ┏ ┓ ┗ ┛ ━ ┃ ┳ ┻ ┣ ┫ ╋
